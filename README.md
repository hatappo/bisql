# Bisql

Bisql (pronunce `bάɪsɪkl` 🚲️ ) is 2-way-sql ORM library for Clojure.

## Development

### Prerequisites

```sh
brew install clojure/tools/clojure
# If you don't use Mac, see https://clojure.org/guides/install_clojure

brew install borkdude/brew/babashka
# If you don't use Mac, see https://github.com/babashka/babashka#installation
```

### Setup

#### Setup Clojure

```sh
clj -P
```

#### Setup PostgreSQL

```sh
docker compose up -d
bb test-db
```

Connection settings:

- host: `localhost`
- port: `5432`
- database: `bisql_dev`
- user: `bisql`
- password: `bisql`

Connect with `psql`:

```sh
psql -h localhost -p 5432 -U bisql -d bisql_dev
```

Quick checks:

```sql
\dn+

\d

SELECT * FROM users;
SELECT * FROM orders;
```

### Generate CRUD SQL Files

Generate CRUD templates from the local PostgreSQL schema and group them into one SQL file per table:

```clj
(require '[bisql.core :as bisql]
         '[next.jdbc :as jdbc])

(def datasource
  (jdbc/get-datasource {:dbtype "postgresql"
                        :host "localhost"
                        :port 5432
                        :dbname "bisql_dev"
                        :user "bisql"
                        :password "bisql"}))

(-> (bisql/generate-crud datasource {:schema "public"})
    (bisql/write-crud-files! {:output-root "resources/sql"}))
```

This writes files such as:

- `resources/sql/postgresql/public/users/users-crud.sql`
- `resources/sql/postgresql/public/orders/orders-crud.sql`

Or run the bundled task:

```sh
bb crud-generate
```
